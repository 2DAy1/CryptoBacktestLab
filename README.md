### README.md

# Backtest Project

## Опис
Проєкт для бектестингу торгових стратегій на основі 1-хвилинних даних Binance. Реалізовано три стратегії: SMA перетин, RSI+BB, VWAP Reversion.

## Вимоги
- Python >= 3.10
- Платформи: Linux / macOS / Windows

## Інструкція по запуску
```bash
pip install -r requirements.txt
python main.py
```

## Стратегії
- **SMACrossStrategy**: перетин короткої і довгої SMA
- **RSIBBStrategy**: вхід по RSI < 30, підтвердження через нижню межу BB
- **VWAPReversionStrategy**: відхилення від VWAP з поверненням

## Результати
- Метрики: `results/` (CSV)
- Графіки: `results/screenshots/`

## Висновки
- **SMA Crossover** — показала стабільну динаміку з невеликою, але позитивною дохідністю. Підходить для спокійного ринку.
- **RSI + BB** — агресивніша стратегія, показала високі піки дохідності, але зі зниженим winrate та вищими просадками.
- **VWAP Reversion** — найкраща за співвідношенням ризику до винагороди, стабільна, з помірним рівнем експозиції. Добре підходить для mean-reversion сценаріїв.

> Загалом усі стратегії демонструють різну поведінку на високочастотних даних. Їх комбінація може покращити загальну ефективність портфеля.


## Архітектура
Архітектура масштабована, легко додавати нові стратегії. Проста інтеграція з VectorBT. Код поділено на модулі, протестовано.
